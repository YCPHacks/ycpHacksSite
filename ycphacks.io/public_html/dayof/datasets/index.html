Data Sets

This page contains data sets provided by the State of Pennsylvania to help you with your project



Department of Transportation

Contains data regarding roads, bridges, railways, construction projects and vehicle accidents.

Commonwealth Open Data

Provides datasets for Governor Wolf's initiatives

Department of Health Data

Provides an interactive health statistics dissemination web tool where you can create customized data tables for various health related data

PASDA

Pennsylvania Spatial Data Access (PASDA) is the official public access geospatial information clearinghouse for the Commonwealth of Pennsylvania.

DCNR

This site provides access to all of our published GIS data, which includes over 130 datasets relative to state parks, state forests, geology, recreational opportunities and more!
